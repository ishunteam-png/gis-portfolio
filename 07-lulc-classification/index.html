About

The full pipeline pulls Sentinel-2 L2A from Element84's free STAC, masks clouds via SCL, builds a Jan–Apr dry-season median composite, derives 13 features (6 bands + NDVI, NDBI, NDWI, MNDWI, BSI, NDMI, BRI), and trains a Random Forest. Predictions are aggregated to a 22×22 grid for this dashboard; the full pipeline writes a per-pixel classified GeoTIFF as well.

Story: Bengaluru built-up grew +22.6% in four years. Bare soil dropped 24% — the visible "field-to-construction" lifecycle. Cropland and vegetation lost ~6% each to absorb that.

Source: Sentinel-2 L2A · Random Forest · Element84 Earth Search STAC · source on GitHub
